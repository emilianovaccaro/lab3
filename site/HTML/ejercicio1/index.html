    PREGUNTAS:

    1 - Ejemplo de uso de etiquetas Meta

    Las etiquetas Meta se utilizan para metadatos, que son datos relacionados al documento pero que no son visualizados en la presentación del mismo. En estas etiquetas yo puedo especificar el tipo de decodificación de datos, puedo pasar links/tags de css para sumar estilos a mi pagina, establecer el SEO de mi pagina para un mejor posicionamiento en busqueda, etc


    2- Atributos aplicables a la etiqueta HTML

    title: Brinda información adicional al usuario cuando deja mantenido el cursor sobre la página
    lang: Define el idioma principal del documento
    entre otros atributos, para mas información se puede visitar la W3School HTML Attributes


    3- ¿Cómo se relaciona un header HTTP con la etiqueta Head de HTML?

    No se pueden enviar headers HTTP desde un HTML, debido a esto, hay un atributo que me permite a mi establecer en una etiqueta meta, el tipo de headers que se van a recibir en una request. Http-equiv (equiv = equivalente) atributo que premite hacer referencia a un header HTTP.


    4- ¿Qué es UTF8?

    UTF-8 es un formato de codificación de caracteres que representa cualquier caracter Unicode. Este formato de codificación permite el uso de tildes y la letra ñ de manera adecuada.


    5- ¿Qué atributo HTML sirve para aplicar estilo y a qué etiqueta es aplicable?

    El atributo style sirve para aplicar estilo al documento y es aplicable a todas las etiquetas que contengan elementos visibles.

    6- ¿Dónde se indica que un enlace debe ubicar el documento en una nueva pestaña?

    atributo="valor" -> target="_blank" en la etiqueta de anclaje o "a" se indica que el enlace deberá ser abierto en una pestaña nueva.

    7- Explicar cuándo se usarían etiquetas semánticas y con qué propósito

    Las etiquetas semánticas se utilizan para la optimización del sitio para los motores de búsqueda. Sirven para enmarcar o estructurar semánticamente el contenido del html dentro del body. La utilización de las etiquetas semánticas generalmente responde a formas estandarizadas de presentar el contenido de una página, como por ejemplo la etiqueta footer suele contener información como direcciones, números de documento, contacto, la etiqueta main indica dónde está la información más relevante del documento, etc. El contenido que haya dentro de cada etiqueta semántica permite que una página tenga una posición más relevante a la hora de filtrar contenido específico en un buscador.

    8- ¿Qué atributos de estilo básicos aplicarían a una tabla? Explicar

    Los atributos de estilo básicos aplicables a una tabla son:

    1. border: Borde de una tabla (sólidos, punteados, de guiones, etc.)
    2. border-collapse: Los bordes de las celdas se combinan en uno solo
    3. width: Indica el ancho de la tabla
    4. height: Indica el alto de la tabla
    5. background-color: Indica el color de fondo de la tabla

    9- ¿Qué diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    La etiqueta BODY de un documento HTML, contendra a todos los elementos visibles para el usuario en un doc html. La etiqueta BODY en una HTTPRequest contendra parte de la información que se esta enviando al servidor que recibe esa request.

    10- ¿Qué atributos marcan la agrupación de celdas de una tabla? ¿A qué etiquetas se aplican?

    El atributo "colspan" me permite indicar cuántas celdas se van a querer agrupar horizontalmente, y el atributo rowspan indica cuántas celdas se van a agrupar verticalmente. Ambas se aplican a las etiquetas td.
